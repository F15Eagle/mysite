## Summary
Lil’ Willy is my high school active-control rocketry team’s (E-Town Rocket Bureau’s) effort to control the apogee of an F-Class model rocket using PID-controlled airbrakes. The project is currently underway, with a target launch window set for the summer of 2026. So far, we’ve conducted one test flight, which—rather unfortunately—ended in a rapid unscheduled disassembly. 

While the rocket was originally intended for the 2026 American Rocketry Challenge, Flight 1’s catastrophic failure sidelined those plans. To learn more about the program and track our progress, you can visit our [website](https://ethsrocketry.github.io/site/arc-2026.html).
